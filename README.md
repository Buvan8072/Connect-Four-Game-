# Connect Four - Python Terminal Game

A simple **Connect Four** game built using Python for the command line interface. Developed as part of an internship project to demonstrate skills in Python, game logic, and user interaction via the terminal.

---

## About the Game

Connect Four is a two-player game where players take turns dropping their respective pieces (`X` and `O`) into a vertical 7-column, 6-row grid. The first player to connect **four pieces in a row** â€” horizontally, vertically, or diagonally â€” wins the game.

---

## Features

- 6x7 grid game board
- Two-player support (Player X and Player O)
- Turn-based gameplay
- Win detection in:
  - Horizontal
  - Vertical
  - Diagonal (both directions)
- Draw detection
- Input validation and error handling

---

## Tech Stack

- Language: **Python 3.x**
- No external libraries required (pure Python)
